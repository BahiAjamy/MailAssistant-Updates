# Assistant Mail — mises à jour

Ce dépôt public héberge uniquement le canal de distribution d’Assistant Mail :

- les images d’installation DMG dans les [Releases](https://github.com/BahiAjamy/MailAssistant-Updates/releases) ;
- le flux Sparkle `appcast.xml` ;
- les notes et sommes de contrôle des versions.

Le code source de l’application reste dans un dépôt privé séparé. Les archives
publiées sont signées avec la clé EdDSA intégrée à l’application. Elles ne sont
pas notarifiées par Apple et sont destinées pour le moment à un usage personnel.
